# 🎬 Multimedia Compressor Pro (Image & Video)

### 🚀 Overview
A professional Desktop Application built with **Python** and **Tkinter** that allows users to compress images and videos efficiently. It features a modern dark-themed GUI and provides granular control over compression quality and bitrates.

---

## ✨ Key Features
- **Dual Support:** Compress both Images (`JPG`) and Videos (`MP4`).
- **Dynamic UI:** The interface automatically switches between "Quality Slider" for images and "Bitrate Slider" for videos.
- **Modern Dark Theme:** A clean, user-friendly interface designed with the `Catppuccin` color palette.
- **Custom Output:** Allows users to define custom names for their processed files.
- **Real-time Processing:** Visual feedback during the compression task to keep the user informed.

---

## 🛠️ Tech Stack
- **GUI Framework:** `Tkinter` (Python's standard GUI library).
- **Image Processing:** `Pillow` (PIL) for advanced image handling and conversion.
- **Video Processing:** `MoviePy` for video editing and bitrate manipulation.
- **System Logic:** `OS` module for directory and path management.

# أَبَا الْقَاسِم — ABA AL-QASIM Heritage Parfumerie 🕌

> موقع ويب لمتجر عطور عربي فاخر، مبني من تصميم Figma إلى كود HTML/CSS/JS كامل.

---

## 📌 نبذة عن المشروع

**أبا القاسم** هو موقع ويب لمتجر عطور عربي أصيل يقدم مجموعة "العود الملكي" الفاخرة.  
المشروع بدأ كتصميم كامل على **Figma** ثم تُرجم يدوياً إلى كود ويب احترافي كمتطلب أكاديمي يجمع بين مهارات التصميم والتطوير.

---

## ✨ مميزات الموقع

- 🎨 تصميم فاخر مستوحى من التراث العربي بألوان ذهبية وداكنة
- 📱 Responsive Design يعمل على جميع أحجام الشاشات
- 🍔 Hamburger Menu موحد في كل الصفحات
- 🔐 صفحات تسجيل دخول وإنشاء حساب مع Validation كامل
- 🛒 عربة تسوق تفاعلية مع حساب المجموع تلقائياً
- 🌟 تأثيرات بصرية: Glassmorphism، Glow، 3D Parallax على الصور
- 🔄 Page Transitions سلسة بين الصفحات
- 📜 Scroll Reveal Animations
- 🍞 Toast Notifications بدلاً من الـ alert

## 🚀 طريقة التشغيل

1. حمّل المشروع أو استنسخه:
```bash
git clone https://github.com/your-username/aba-al-qasim.git
```

2. افتح ملف `HTML/Home.html` في المتصفح مباشرةً — لا يحتاج server أو تثبيت أي شيء.

## 🛠️ التقنيات المستخدمة

| التقنية | الاستخدام |
|--------|-----------|
| **HTML5** | هيكل الصفحات مع دعم RTL كامل |
| **CSS3** | تصميم، Animations، Responsive، Glassmorphism |
| **JavaScript (Vanilla)** | تفاعلية الصفحات بدون أي Framework |
| **Google Fonts** | خط Amiri للطابع العربي الأصيل |
| **Font Awesome 6** | أيقونات التواصل الاجتماعي والواجهة |
| **Figma** | التصميم الأولي الكامل قبل التكويد |

## 🎨 من Figma إلى Code

المشروع بدأ بتصميم كامل على **Figma** يشمل:
- تصميم كل صفحة بالتفصيل (Wireframe ثم UI كامل)
- اختيار نظام الألوان الذهبي والداكن
- تحديد الـ Typography (خط Amiri)
- تصميم المكونات (Cards، Buttons، Navbar)

ثم تُرجم التصميم يدوياً إلى كود مع إضافة تأثيرات تفاعلية لم تكن موجودة في الـ Figma.
